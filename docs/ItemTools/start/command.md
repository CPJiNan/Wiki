# 🎼 命令

```
<> - 必填项 [] - 选填项

/itemtools - 插件主命令

/item list - 查看物品列表
/item get <物品 ID> [数量] - 获取物品
/item give <玩家名称> <物品 ID> [数量] - 给予玩家物品
/item save [物品 ID] [保存路径] - 保存物品到配置

/loreedit check - 查看手中物品 Lore
/loreedit add <内容> - 为手中物品新增 Lore
/loreedit remove <行数> - 为手中物品移除 Lore
/loreedit set <行数> <内容> - 为手中物品设置 Lore
/loreedit insert <行数> <内容> - 在指定行后面插入 Lore
/loreedit clear - 清空手中物品的全部 Lore

/nbtedit check - 查看手中物品 NBT
/nbtedit set <键> <值> - 设置手中物品 NBT

/enchant check - 查看手中物品附魔
/enchant set <附魔类型> <附魔等级> - 设置手中物品附魔

/unbreakable true - 设置手中物品无法破坏
/unbreakable false - 移除手中物品无法破坏

/itemtools reload - 重载插件配置
```
