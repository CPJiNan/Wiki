# 🎼 命令

```
/akarilevel - 插件主命令 (Aliases: /exp 或 /level)

/akarilevel exp add [玩家名称] [数值] - 给予玩家指定数值的经验
/akarilevel exp remove [玩家名称] [数值] - 移除玩家指定数值的经验
/akarilevel exp set [玩家名称] [数值] - 设置玩家指定数值的经验
/akarilevel exp check [玩家名称] - 查询玩家经验

/akarilevel level add [玩家名称] [数值] - 给予玩家指定数值的等级
/akarilevel level remove [玩家名称] [数值] - 移除玩家指定数值的等级
/akarilevel level set [玩家名称] [数值] - 设置玩家指定数值的等级
/akarilevel level check [玩家名称] - 查询玩家等级

/akarilevel levelup - 玩家请求升级

/akarilevel reload - 重载插件配置(settings.yml|level.yml)
```
