# 🏷 变量
```
%AkariLevel_Level% - 玩家当前等级
%AkariLevel_LastLevel% - 玩家当前等级的上一个等级
%AkariLevel_NextLevel% - 玩家当前等级的下一个等级

%AkariLevel_Exp% - 玩家当前经验值

%AkariLevel_LevelName% - 玩家当前等级名称
%AkariLevel_LastLevelName% - 玩家当前等级上一个等级的名称
%AkariLevel_NextLevelName% - 玩家当前等级下一个等级的名称

%AkariLevel_LevelExp% - 升级到玩家当前等级所需经验值
%AkariLevel_LastLevelExp% - 升级到玩家当前等级的上一个等级所需经验值
%AkariLevel_NextLevelExp% - 升级到玩家当前等级的下一个等级所需经验值

%AkariLevel_ExpProgressBar% - 玩家当前经验 / 升至下一级所需经验 的进度条
%AkariLevel_LevelProgressBar% - 玩家当前等级 / 最高等级 的进度条
```
